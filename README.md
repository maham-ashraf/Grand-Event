**Grand Event**

Grand Event is a simple event management website designed to showcase event details in a clean and organized way.  
This project was built to practice front-end development and understand how event-based websites are structured.


**Features**

- Event details display  
- Clean and attractive layout  
- Simple navigation  
- Basic responsive design  
- Beginner-friendly structure  

**Tech Stack**

- HTML  
- CSS  
- JavaScript
